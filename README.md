jekyll-website
===============

New version of the website using jekyll, gulp, font-awesome and bootstrap

Prerequisites
-------------

* node (http://richardhsu.net/2013/10/19/installing-nodejs-npm-on-ubuntu-13-10/)
* bower
* gulp
* jekyll

OR




Install
-------

    npm install
    bower install

Run with drafts
----

    gulp server --drafts

Run like prod without drafts
----

	gulp server --prod
